# 🧠 Mindora Psychology Platform

A modern, scalable, and fully dynamic psychology management platform designed for mental health professionals and clinics. Built with ASP.NET Core Web API and React, Mindora provides a seamless experience for both administrators and visitors through a powerful content management system and a responsive user interface.

## 🚀 Overview

Mindora is more than a simple psychology website. It is a full-stack platform that enables psychologists and clinics to manage their online presence, publish content, receive appointment requests, and control website data through a centralized administration panel.

This project was developed with clean architecture principles, scalability, and maintainability in mind.
